# Financial Modeling Asset Comparison Project

This repository contains an Excel-based asset comparison project covering multiple asset classes including:

- Gold
- Silver
- Crude Oil
- Natural Gas
- USD/INR
- Bank Nifty
- Copper
- Zinc
- Soya

## What This Project Includes

The Excel workbook includes:

- Historical asset price data
- Annual return comparison
- VAMI analysis
- Asset performance comparison
- Commodity and market index analysis

## Who This Is Useful For

This project is useful for:

- Finance students
- Financial modeling learners
- Equity research aspirants
- Investment banking aspirants
- Portfolio analysis learners
- Excel modeling practice
